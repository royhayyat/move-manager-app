# 🏠 מנהל המעבר החכם (Moving Manager)

אפליקציית ווב (Single Page Application) לניהול תקציב ומשימות מעבר דירה בקלות ובכיף.
האפליקציה נבנתה בדגש על חווית משתמש (UI/UX), עבודה ללא שרת, ושמירת נתונים מקומית.

## ✨ פיצ'רים עיקריים

* **שמירה אוטומטית:** כל הנתונים נשמרים ב-LocalStorage בדפדפן. לא הולך לאיבוד בריענון.
* **ניהול מסמכים:** צירוף קבלות ותעודות אחריות (נשמר ב-IndexedDB מקומי).
* **דאשבורד פיננסי:** חישוב בזמן אמת של תקציב משוער מול בפועל, חריגות וחיסכון.
* **גיימיפיקציה:** מד התקדמות שמתמלא ככל שרוכשים יותר פריטים.
* **ייצוא נתונים:** אפשרות לייצא לאקסל (CSV) או להדפיס דוח מסודר.
* **עיצוב רספונסיבי:** עובד מעולה גם במובייל וגם במחשב.

## 🛠️ טכנולוגיות

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
* **Design:** Bootstrap 5 + התאמות CSS אישיות (Glassmorphism).
* **Storage:** LocalStorage (Metadata), IndexedDB (Files).
* **Fonts & Icons:** Google Fonts (Rubik), FontAwesome.

## 🚀 איך להשתמש?

1.  הכנסו לקישור הדמו (למטה).
2.  הוסיפו פריטים לרשימה (ריהוט, מוצרי חשמל וכו').
3.  הגדירו תקציב משוער.
4.  לאחר רכישה - סמנו V, עדכנו מחיר אמיתי וצרפו צילום קבלה.
